## Projektstruktur

Die wichtigsten Ordner im Repository:

- **assets** – Medien wie Bilder und Videos
- **docs** – Dokumentationen des Repositories
- **documents** – Interne Dokumente der Black Market GmbH
- **secrets** – Sensible Daten (nicht öffentlich zugänglich)
- **sources** – Quellcode der Projekte
- **website** – Code der Firmenwebsite
